# ⛽ Vehicle Service & Fuel Station Management System

A web-based management system developed to streamline vehicle servicing operations and fuel station management. The platform enables customers, mechanics, service managers, and fuel managers to efficiently manage vehicle services, fuel inventory, bookings, and customer information through a centralized system.

---

## 📌 Project Overview

The Vehicle Service & Fuel Station Management System is designed to improve operational efficiency by digitizing service booking, vehicle management, customer records, fuel station operations, and administrative tasks.

## ⚙️ Installation Guide

### Clone Repository

```bash
git clone https://github.com/pabodhainduwari/Vehicle-Service-And-Fuel-Station-Management-System.git
```

### Configure Database

1. Create MySQL Database
2. Import SQL Scripts
3. Update Database Credentials

### Run Application

1. Import project into Eclipse
2. Configure Apache Tomcat Server
3. Run on Server

```text
http://localhost:8080/VehicleService_webSystem
```
